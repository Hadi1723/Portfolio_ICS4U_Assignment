Top 10 Modules

1. Algorithms

The only project required for this module was the Algorithm Exploration assignment, in which I had to explore the algorithm required to solve cubes. Ultimately, I was able to solve multiple cubes faster as I started understanding the steps, also known as algorithms, required to solve the problems (which was to solve the cubes). Upon finishing the assignment, I realized that the idea of algorithms helped me to better complete coding assignments in this course. For example, I learned that by laying out a step of how my programs will function before trying to code, my efficiency to complete my assignments increased. Also, laying out the structure of my program allows me to better understand the structure of my program better, this made it easier for me to make changes since I had better judgement on which places could use adjustments. I will use the concepts that I learned in this course to explore other benefits of algorithms such as its ability to improve the speed of a program by using less computer memory.

In this module, I learned how JavaScript programming can be used to model real-life actions or behaviour. More specifically, I learned how JavaScript can incorporate common concepts in Physics such as Vectors (the application of more than 1 dimensions), Forces, kinematics (velocity and acceleration), behaviour of objects, and more. This allowed me to model realistic motion of objects and simulate events that would occur in real life, as fish reaction to predators (as evident in the project Creature Colonies) . The knowledge that I learned and applied in this course will help me in my career because I can use these skills to help me code programs most useful in real-life applications, such as simulations that can predict weather conditions.
